Password Security FAQ

Is this password checker secure and private?

Yes, 100% private. All strength analysis happens in your browser. For the breach check, we use a technique called "k-Anonymity" where only the first 5 characters of a hashed (scrambled) version of your password are sent. Your full password never leaves your device, ensuring complete privacy.

What does "exposed in a data breach" mean?

It means the password you entered was found in one or more massive data leaks from compromised websites. Hackers use these lists to try and log into other accounts. If your password is on such a list, you should stop using it immediately, everywhere.

What is "Time to Crack"?

This is an estimate of how long it would take an average desktop computer to guess your password using common "brute-force" attack methods. A longer time to crack indicates a much stronger password.
